Для создания ветки применяется команда git branch <имя_ветки> Для создания ветки и сразу перехода на неё используется команда 
Для переключения между ветками/коммитами используется команда git checkout <имя_ветки>/<id_коммита>
Для изменения текста коммита используется команда git commit --amend -m "новый текст комита"
Чтобы запушить новую ветку на Хаб используется комманда git puch origin <имя_ветки>
Для возврата к предыдущему состоянию файла(предыдущему коммиту) применяется комманда git reset <имя_файла>/<id_коммита>
